# **`Lambda` in Python** 

## üîπ What is a `lambda` function?

A `lambda` is a **short, anonymous function** defined with the `lambda` keyword.
It‚Äôs mainly used for **quick, throwaway functions** ‚Äî especially when passing logic as a parameter.

---

### ‚úÖ Syntax:

```python
lambda arguments: expression
```

---

## ‚úÖ Example 1: Double a number (like unit conversion)

üî∏ Use case: You want to quickly double a number (like converting meters to centimeters)

```python
double = lambda x: x * 2
print(double(5))  # Output: 10
```

---

## ‚úÖ Example 2: Sort names by their length

üî∏ Real use: Sort a list of names based on how long the name is.

```python
names = ["Deepak", "Ankit", "Amitabh", "Raj"]
sorted_names = sorted(names, key=lambda name: len(name))
print(sorted_names)  # Output: ['Raj', 'Ankit', 'Deepak', 'Amitabh']
```

---

## ‚úÖ Example 3: Filter even numbers

üî∏ Real use: Select even numbers from a list (e.g., filtering invoice numbers)

```python
numbers = [1, 2, 3, 4, 5, 6]
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # Output: [2, 4, 6]
```

---

## ‚úÖ Example 4: Apply tax to prices

üî∏ Real use: You're given a list of prices, and you want to apply a 10% tax.

```python
prices = [100, 200, 300]
final_prices = list(map(lambda x: x * 1.1, prices))
print(final_prices)  # Output: [110.0, 220.0, 330.0]
```

---

## ‚úÖ Example 5: Ternary logic using lambda (pass/fail)

üî∏ Real use: Labeling test scores.

```python
check_pass = lambda score: "Pass" if score >= 50 else "Fail"
print(check_pass(75))  # Output: Pass
print(check_pass(35))  # Output: Fail
```

---

## ‚úÖ When to use `lambda`

| Situation                        | Use `lambda`? |
| -------------------------------- | ------------- |
| Short, one-liner function        | ‚úÖ Yes         |
| Function used only once          | ‚úÖ Yes         |
| You need to name and reuse logic | ‚ùå Use `def`   |
