# Dictionary Comprehension
Here are some practical and clean **dictionary comprehension examples** in Python ‚Äî a powerful way to create or transform dictionaries in a single line.

---

### ‚úÖ 1. **Basic Syntax**

```python
{key: value for item in iterable}
```

---

### ‚úÖ 2. **Squares of Numbers**

```python
squares = {x: x**2 for x in range(1, 6)}
print(squares)
```

#### üñ®Ô∏è Output:

```python
{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

---

### ‚úÖ 3. **Uppercase Keys**

```python
data = {'env': 'dev', 'owner': 'deepak'}
upper_keys = {k.upper(): v for k, v in data.items()}
print(upper_keys)
```

#### üñ®Ô∏è Output:

```python
{'ENV': 'dev', 'OWNER': 'deepak'}
```

---

### ‚úÖ 4. **Filter Items in a Dict (e.g., only values > 50)**

```python
marks = {'Math': 90, 'English': 45, 'Science': 80}
passed = {k: v for k, v in marks.items() if v >= 50}
print(passed)
```

#### üñ®Ô∏è Output:

```python
{'Math': 90, 'Science': 80}
```

---

### ‚úÖ 5. **Swap Keys and Values**

```python
original = {'a': 1, 'b': 2, 'c': 3}
swapped = {v: k for k, v in original.items()}
print(swapped)
```

#### üñ®Ô∏è Output:

```python
{1: 'a', 2: 'b', 3: 'c'}
```

---

### ‚úÖ 6. **Create Tag Format for AWS**

```python
tags = {'Environment': 'dev', 'Owner': 'Deepak'}
aws_tags = [{'Key': k, 'Value': v} for k, v in tags.items()]
print(aws_tags)
```

#### üñ®Ô∏è Output:

```python
[{'Key': 'Environment', 'Value': 'dev'}, {'Key': 'Owner', 'Value': 'Deepak'}]
```

---

### ‚úÖ 7. **Word Lengths from a Sentence**

```python
sentence = "DevOps is awesome"
length_map = {word: len(word) for word in sentence.split()}
print(length_map)
```

#### üñ®Ô∏è Output:

```python
{'DevOps': 6, 'is': 2, 'awesome': 7}
```
